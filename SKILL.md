#### Step 3.1：调用 quant_analyzer_v22.py 获取量化数据

**必须执行的命令：**
```bash
cd /root/.openclaw/workspace/skills/xiaohuo-quant-combined/scripts
/root/.openclaw/workspace/skills/xiaohuo-quant-combined/scripts/venv/bin/python quant_analyzer_v22.py --code {股票代码} --days 90 --output {股票代码}_result.json
```

**新特性说明（v2.2）：**
- **SQLite 本地缓存优先：已有的数据直接从数据库读取，速度提升 1.5 倍
- **双数据源自动互补：
  - 首选 Tushare（字段更丰富）
  - Tushare 失败时自动降级到 AkShare（免费无限制）
  - 自动缓存新数据到 SQLite
- **无需强制指定数据源：系统自动选择最优方案

**分析天数默认90天**
**结果保存到 `{股票代码}_result.json`**

#### Step 3.2：读取分析结果

读取生成的 JSON 文件，提取以下数据：
- 股票代码、股票名称
- 分析时间、数据范围
- 最新行情：最新价、涨跌额、涨跌幅、成交量、成交额
- 技术指标：MA5、MA10、MA20、MA60、DIF、DEA、MACD、RSI(14)
- 策略信号：信号总数、买入信号数、卖出信号数、最近10条信号
- 回测结果：初始资金、最终资产、总收益率、最大回撤、总交易次数、买入次数、卖出次数、胜率、交易记录
