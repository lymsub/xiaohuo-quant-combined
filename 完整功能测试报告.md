### Bug 2: AkShare股票代码格式 ✅ 已修复

**问题描述**:
- `_convert_from_ts_code` 只返回 `600519`
- AkShare需要完整格式 `sh600519`

**影响范围**:
- AkShare数据源调用
- 可能导致数据获取失败

**修复方案**:
```python
# 修复前
def _convert_from_ts_code(self, ts_code: str) -> str:
    if '.' in ts_code:
        return ts_code.split('.')[0]  # ❌ 只返回 "600519"
    return ts_code

# 修复后
def _convert_from_ts_code(self, ts_code: str) -> str:
    if '.' in ts_code:
        code, market = ts_code.split('.')
        if market == 'SH':
            return f"sh{code}"  # ✅ 返回 "sh600519"
        elif market == 'SZ':
            return f"sz{code}"  # ✅ 返回 "sz000001"
        elif market == 'BJ':
            return f"bj{code}"  # ✅ 返回 "bj800000"
    return ts_code
```
